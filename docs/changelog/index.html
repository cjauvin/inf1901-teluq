Évolution du cours

Évolution du cours #

L’intelligence artificielle est un sujet qui évolue à une vitesse prodigieuse, et le cours INF 1901 tente donc d’évoluer lui aussi, avec des changements continuels.

2025-11-24 : Ajout d’une section au module 2, sur l’algorithme des plus proches voisins et du dilemme biais-variance.

2025-11-17 : Ajout d’architectures neuronales avancées au module 3 : les autoencodeurs, les réseaux de neurones pour les graphes ainsi que les machines de Turing neuronales.
